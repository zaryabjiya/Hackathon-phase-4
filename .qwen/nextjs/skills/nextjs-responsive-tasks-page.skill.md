# Next.js Responsive Tasks Page Skill

## Description
This skill creates a responsive Next.js tasks page with proper layout, styling, and functionality. It implements a responsive design using Tailwind CSS that works well on different screen sizes.

## Purpose
- Create a responsive tasks page in Next.js using App Router
- Implement proper layout and component structure
- Use Tailwind CSS for responsive styling
- Include task listing, creation, and management functionality
- Ensure mobile-first responsive design
- Output complete files with appropriate paths
- Follow Next.js 16+ App Router conventions

## Implementation Requirements
- Use Next.js App Router (app directory structure)
- Implement responsive layout with Tailwind CSS
- Create components for task cards, forms, filters, etc.
- Include client-side interactivity where needed
- Implement proper TypeScript typing
- Add loading and error states
- Include accessibility features
- Use appropriate Next.js hooks and patterns
- Follow Next.js best practices for performance

## Output Format
- Complete file content for the tasks page
- File path: frontend/src/app/tasks/page.tsx
- Additional component files as needed
- Include proper imports and component structure
- Add responsive design classes with Tailwind
- Include TypeScript interfaces for props and data

## Additional Considerations
- Implement proper error boundaries
- Add loading skeletons for better UX
- Include keyboard navigation support
- Optimize for performance with lazy loading if needed
- Follow accessibility best practices (WCAG)
- Ensure touch-friendly interface elements
- Use responsive breakpoints appropriately
- Implement proper form validation
